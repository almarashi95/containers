# Containers for the Cummings-McCabe Research Group
This repository serves as a place for members of our research group to store, share, and have version-control for containerized software environments.

## Adding a new container:
1) Make a new branch or fork of this repository.
2) Make a new directory with a descriptive name in `dockerfiles/`.
2) Place your Dockerfile that builds the container and a README with context and information about the container.
3) Make a pull-request to add this directory and its contents to the `main` branch.
